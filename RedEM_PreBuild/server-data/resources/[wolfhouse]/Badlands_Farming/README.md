# WolfHouse RedEM:RP Farming script

## 1. Requirements
[redemrp_notification]

[redemrp_inventory]

[redemrp_progressbars]

## 2. Setup

- Add ```ensure Badlands_Farming``` in server.cfg/resource.cfg

- If your inventory does not close after use, add this to the client.lua of redemrp_inventory
```lua
RegisterNetEvent('redemrp_inventory:close_inventory')
AddEventHandler('redemrp_inventory:close_inventory', function()
    SendNUIMessage({type = 'close'})
    SetNuiFocus(false, false)
    isInventoryOpen = false
    isOtherOpen = false
end)
```
